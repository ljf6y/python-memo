障害内容

クレンジング処理にて、前後空白のみ削除すべき項目について、
文字列中の半角スペースまで削除していた。
その結果、本来チェック対象とすべき中間空白が除去され、
入力チェックが正しく行われない事象が発生した。

原因区分

設計不備

原因内容

前後空白削除のみを想定していたが、
実装時に replace(" ", "") を用いたため、
文字列中の半角スペースも削除する処理となっていた。


対策内容



① クレンジング処理を修正し、前後空白のみ削除するよう変更した。
② 文字列中の半角スペースは削除せず、入力チェック対象として扱うよう修正した。
③ 中間空白を含むデータのUT観点を追加した。
