1.4 データチェック処理を行う。

編集後アップロード情報に対してデータチェックを実施する。  
データチェックは共通チェック部品を利用して行う。

err_msg_list = 共通チェック部品.validate_dataframe(upload_df_after, チェックルール)

※データチェックの詳細仕様については、  
外部設計書「5-04-05-0 入出力設計書_アップロード（不動産粗利）.xlsx」の  
「個別処理（データチェック）」シートに記載の  
「チェック仕様（データ）」を参照すること。

※チェックルールの定義方法については、  
内部設計書「プログラム機能書（チェック）.xlsx」の  
「メソッド仕様書_DFチェック」シートの  
「処理詳細」に記載された「■メソッド引数」を参照すること。
